Replicator

Set up a node to replicate its own feed and any feeds it subscribes to Makes use of Hyperswarm to locate other peers who have or are interested in the feeds this node has Currently this creates a swarm instance per foreign feed and does not reuse connections. This can probably be improved upon in the future using multiplexing
